# CSLOL - Créateur de champion League Of Legends

Une application **Avalonia (C#)** permettant de créer et personnaliser un champion de League of Legends.

Points clés du projet :

- Programmation Orientée Objet : Utilisation de l'héritage et du polymorphisme pour gérer les synergies entre les races (humains, yordles, etc.) et les classes (Mage, Tank, etc.).
- UI Dynamique : Interface réactive où chaque sélection de champion ou de classe met à jour instantanément les statistiques, compétences et visuels.
- Intégration API : Utilisation de l'API officielle Riot Games DataDragon pour générer un inventaire aléatoire (items, icônes et descriptions) en temps réel.
- Expérience Utilisateur : Design immersif avec arrière-plan personnalisé, gestion de la transparence et fenêtre de résumé final structurée.
